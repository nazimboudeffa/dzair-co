À propos du projet

dzair.co est un annuaire des sites institutionnels algériens : présidence, gouvernement, ministères, wilayas, universités et organismes publics.

L'objectif est de centraliser et maintenir des liens fiables vers les portails officiels de l'État algérien, pour faciliter l'accès des citoyens aux services en ligne.

Sous-domaines dzair.co

Comment utiliser l'annuaire

  1. Utilisez la barre de recherche en haut de la page pour filtrer les sites par nom ou URL.
  2. Parcourez les catégories (Présidence, Ministères, Wilayas…) via le sommaire.
  3. Cliquez directement sur un lien pour ouvrir le site officiel dans un nouvel onglet.
  4. Les sites archivés ou temporairement hors ligne sont regroupés dans la section Archive en bas de page.

Contribuer – ajouter ou corriger un lien

L'annuaire est maintenu sur GitHub. Si vous souhaitez ajouter un site, corriger une URL ou signaler un lien mort, vous pouvez éditer directement le fichier index.html et soumettre une Pull Request.

  1. Ouvrez le dépôt GitHub ci-dessous.
  2. Cliquez sur index.html puis sur l'icône ✏️ Edit this file.
  3. Apportez vos modifications et soumettez une Pull Request.
